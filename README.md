# FluxEdit

A lightweight, single-file browser-based video editor for Windows.

## Features

- **Timeline editing** — drag, trim, and arrange video, image, audio, and text clips on a multi-track timeline
- **Zoom effects** — add Ken Burns-style zoom animations with three modes:
- **Animate** — smoothly zoom from one level to another over the clip duration
- **Hold** — instantly lock to a fixed zoom level for the entire clip
- **Ramp then Hold** — animate for a set duration, then hold at the final zoom level
- **Zoom easing** — linear, ease-in, ease-out, or ease-in-out curves
- **Text overlays** — add and style text on top of your footage
- **Audio tracks** — layer background music or sound effects
- **Light adjustments** — brightness, contrast, saturation per clip
- **Render to video** — export your project using the browser's MediaRecorder API
- **Save / Load** — projects save to a JSON file and reload with full fidelity
- **No internet required** — runs entirely offline

## Requirements

- Windows 10 or 11

## Installation

Download `FluxEdit_Setup.exe` from the [Releases](../../releases) page and run it.

A desktop shortcut is created automatically. Double-click it to launch.
